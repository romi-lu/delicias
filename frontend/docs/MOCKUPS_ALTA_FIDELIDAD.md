# Mockups en Alta Fidelidad - Delicias

## Descripción de Pantallas

### 1. Home / Landing

**Hero**
- Imagen de fondo a pantalla completa (banner panadería)
- Overlay gradiente blanco de izquierda a derecha (95% → 60% opacidad)
- Badge superior: "Panadería artesanal desde Huancayo" con icono Sparkles
- Headline en dos líneas: "Recién horneado, / hecho para ti"
- Subheadline descriptivo
- CTAs: "Ordenar Ahora" (primario) + "Ver Menú" (outline)
- Trust badges: Envío gratis, Horneado diario, Ingredientes naturales

**Features (¿Por qué elegirnos?)**
- 3 cards en grid
- Imagen 4:3 con overlay gradiente
- Badge con icono sobre imagen (Wheat, Cake, Gift)
- Texto descriptivo + link "Ver más" con flecha

**Nosotros**
- Layout 2 columnas (texto | imagen)
- Lista con checkmarks verdes
- CTAs: Ver menú, Contacto

**Visítanos**
- Mapa embebido + galería de productos en grid 2x3

**Productos destacados**
- Grid de ProductCards (2-5 columnas según viewport)
- Cada card: imagen, badges, nombre, precio, botones

**Testimonios**
- 3 cards con estrellas, cita, nombre

**Contacto**
- Formulario centrado en card
- Campos: nombre, email, teléfono, mensaje
- Botón "Enviar" con icono

---

### 2. Navbar

- **Top bar**: Envío gratis, teléfono, enlaces (Contáctanos, Ordenar)
- **Nav principal**: Logo, links (Inicio, Menú, Historial, Nosotros, Contacto), buscador, CTA Ordenar, carrito, theme toggle, auth
- **Barra categorías**: Chips horizontales scroll
- **Móvil**: Menú hamburguesa, panel desplegable

---

### 3. Footer

- 4 columnas: Brand + logo, Enlaces, Contacto, Redes
- Bottom bar: Copyright, Términos, Privacidad, Contacto
- Icono corazón en "Hecho con ❤️ en Huancayo"

---

### 4. Productos

- Sidebar filtros (sticky): búsqueda, categoría, precio min/max, checkboxes
- Grid de ProductCards
- Paginación
- Chips de filtros activos

---

### 5. Checkout

- Resumen de productos (lista con imágenes)
- Formulario: fecha, dirección, teléfono, método de pago, comprobante
- Sidebar ayuda
- Botón "Confirmar pedido"

---

### 6. Login / Register

- Card centrada
- Campos con labels
- Botón primario
- Link a registro/login

---

## Prototipo Interactivo

El frontend actual **es** el prototipo interactivo. Ejecutar:

```bash
cd frontend && npm run dev
```

Abrir http://localhost:6001 para navegar todas las pantallas.
